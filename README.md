# Secure Authentication System

A backend authentication system built using **Spring Boot** and **Spring Security** implementing **JWT-based authentication**.

## Tech Stack

* Java
* Spring Boot
* Spring Security
* JWT
* MySQL
* Maven

## Features

* User Registration
* Secure Login
* Password Encryption
* JWT Token Generation
* Protected APIs
* Stateless Authentication

## API Endpoints

POST /auth/register
POST /auth/login
GET /api/hello (Protected API)

## Security Implementation

* Spring Security configuration
* JWT filter for request authentication
* Password hashing using BCrypt
* Stateless session management
